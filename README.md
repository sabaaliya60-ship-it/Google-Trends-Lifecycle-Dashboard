# 📈 Google Trends Topic Lifecycle Dashboard

## 📌 Project Overview

This project analyzes the complete lifecycle of Google Trends topics.

It helps answer:

- When did a trend start?
- When did it reach peak popularity?
- How many times did interest drop?
- How many recoveries happened?
- What are the future trend possibilities?

---

## 🛠 Tech Stack

- Python
- Pandas
- Streamlit
- Plotly
- Scikit-Learn

---

## 📊 Dashboard Features

- 8 KPI Cards
- Interest Timeline
- Monthly Trend
- Peak vs Drop Analysis
- Recovery Analysis
- Lifecycle Distribution
- Region Analysis
- Weekly Trend
- Future Trend Prediction
